# Bulletin Store Schema (Demo)

_Created 2026-06-14. Stand-in for Hyperledger Fabric chaincode while Docker/Go are unavailable on the dev machine. Replace with `saksi-bulletin/client-sdk` calls once Fabric is up._

## Overview

The four BalotaChain apps share state via a single JSON file. The file is the source of truth for the one-voter / one-election staging demo.

- **Path (all platforms):** `$HOME/.balotachain/bulletin.json`
  - macOS: `/Users/<user>/.balotachain/bulletin.json`
  - Linux: `/home/<user>/.balotachain/bulletin.json`
  - Windows: `C:\Users\<user>\.balotachain\bulletin.json`
- **Encoding:** UTF-8 JSON, pretty-printed (2-space indent) so a human can inspect it.
- **Concurrency:** demo only; last-writer-wins. Each app rewrites the entire file on every mutation.
- **Atomicity:** apps must write to `bulletin.json.tmp` then `rename` to `bulletin.json` so a crash mid-write doesn't corrupt the store.

## Empty file

When `bulletin.json` does not exist or is empty, apps treat it as:

```json
{
  "version": 1,
  "election": null,
  "voters": [],
  "credentials": [],
  "ballots": [],
  "partial_decryptions": [],
  "tally": null
}
```

## Roles per app

| App | Writes | Reads |
|---|---|---|
| `admin`   | `election`, `voters`, `credentials` | — |
| `voter`   | `ballots` (append)                  | `election`, `credentials` |
| `trustee` | `partial_decryptions` (append), `tally` (when threshold met) | `election`, `ballots`, `partial_decryptions` |
| `auditor` | (read-only)                         | everything |

## Tracking code

`BC-XXXX-XXXX` uppercase hex; 4+4 hex chars; client-generated by voter app via cryptographically secure RNG; uniqueness checked against existing `ballots[].tracking_code` on submit (retry on collision).

## Fingerprint

`sha256:<hex>` of the canonically serialised `tally.results` object — sort keys alphabetically, no whitespace. All apps must use the same canonical algorithm.

## Migration to Fabric

Each `bulletin-store.*.ts/dart/rs` adapter exposes a thin interface:

```ts
interface BulletinStore {
  load(): Promise<Bulletin>;
  save(b: Bulletin): Promise<void>;
}
```

To switch to Fabric, replace the file-backed adapter with one that calls `saksi-bulletin/client-sdk` over gRPC. Apps don't change.
